Αλέξανδρος Ζάντζας


Λίγα λόγια :

-Ο Αλέξανδρος Ζάντζας έχει κατάγωγη από τη Λιβαδειά Βοιωτίας και σπουδαζει στη σχολή Μηχανικών Πληροφορικής της Καβάλας.Είναι 19 χρονών και βρίσκεται στο 2ο ετος των σπουδών του.

Βιογραφικό :

-Κατέχει βασικές γνώσεις προγραματισμού και ιδιαίτερα σε Java, Java FX, C++, Python, HTML, Oracle SQL.Επίσης έχει εργαστεί σε διάφορα project εφαρμογών αποκτόντας εμπειρία.Επιπλέον έχει πτυχίο επιπέδου lower στα αγγλικά.

Χόμπυ :

-Παίζει σε ομάδα ποδοσφαίρου στην θεση αμυντικού και παράλληλα στον ελεύθερο του χρόνο γυμνάζεται μόνος του για την καλύτερη φυσική κατάσταση του .
Εμμανουήλ Δρόσος
Λίγα λόγια :
- Ο Εμμανουήλ Δρόσος έχει κατάγωγη από τη Χρυσούπολη Καβάλας και σπουδαζει στη σχολή Μηχανικών Πληροφορικής της Καβάλας.Είναι 20 χρονών και βρίσκεται στο 2ο ετος των σπουδών του.

Βιογραφικό :

-Κατέχει βασικές γνώσεις προγραματισμού και ιδιαίτερα σε Java, Java FX, C++, Python, HTML, Oracle SQL.Επίσης έχει εργαστεί σε διάφορα project εφαρμογών αποκτόντας εμπειρία.Ακόμη έχει πτυχίο επιπέδου β2 στα γερμανικά και επιπέδου advanced στα αγγλικά.

Χόμπυ :

-Επίσης παίζει σε ομάδα ποδοσφαίρου στην θεση πλάγιου επιθετικού και του αρέσει παράλληλα στον ελεύθερο του χρόνο να γυμνάζεται και εκείνος όπως συνηθίζουν να κάνουν και περισσότεροι άλλοι ποδοσφαιριστές.
